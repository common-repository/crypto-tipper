== Description ==
Allow your authors to request tips via crypto currency on their posts

== Installation ==
1. Upload the `crypto-tipper` folder to your `/wp-content/plugins/` directory.

2. Activate the "Crypto Tipper" plugin in your WordPress administration interface.

3. Edit your profile to select a crypto currency and add your currency ID

== Screenshots ==
1. Each author can set their own crypto currency 
2. Each post you write has your currency and ID displayed
